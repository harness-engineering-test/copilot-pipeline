# operator-frontend

オペレーター向け管理フロントエンド（React + TypeScript + Vite）

## 開発環境

### セットアップ

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

### テスト実行

```bash
npm test
```

### Lint

```bash
npm run lint
```

## ディレクトリ構成

```
src/
  api/         # API クライアント
  components/  # 共有コンポーネント
  hooks/       # カスタムフック
  pages/       # ページコンポーネント
  main.tsx     # エントリーポイント
  App.tsx      # ルート定義
```
